# HTML PPT Studio 初始化配置指南

> **目标**：让零基础新手在 5 分钟内完成项目配置，之后只需把这份文档丢给 AI 工具，AI 就能帮你生成精美 PPT。
> 
> **你需要准备**：一个豆包（Doubao / 火山引擎）API Key

---

## 📋 前置要求

在开始之前，请确保你的电脑已安装：

| 软件 | 用途 | 下载地址 |
|------|------|----------|
| **Python 3.8+** | 运行 AI 图片生成脚本 | https://www.python.org/downloads/ |
| **Git** | 克隆项目（可选，也可直接下载 ZIP） | https://git-scm.com/downloads |
| **Chrome 浏览器** | 预览 PPT & 导出 PNG（可选但强烈建议） | https://www.google.com/chrome/ |

**验证 Python 是否安装成功**：
```bash
python --version
# 或
python3 --version
```
看到版本号（如 `Python 3.10.11`）即表示安装成功。

---

## 🔑 第一步：获取 API Key

本项目使用**豆包（Doubao）Seedream API** 生成 AI 图片，你需要一个火山引擎的 API Key。

### 获取步骤：

1. 打开 https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey
2. 登录火山引擎账号（没有就注册一个）
3. 点击「创建 API Key」
4. 复制生成的 Key（形如 `1a8b8f54-7e25-4138-9542-b7465b47791b`）
5. **妥善保存**，不要泄露给他人

> 💡 **新用户福利**：火山引擎通常有免费额度，足够生成数十张图片。

## 📁 第二步：获取项目代码

```bash
git clone <项目仓库地址> html-ppt-skill

```

1. 下载项目 ZIP 包并解压
2. 进入解压后的文件夹 `html-ppt-skill/`

---

## ⚙️ 第三步：配置 API Key

在项目根目录下找到 `.env` 文件，打开它，把你刚才复制的 API Key 粘贴进去：

```bash
# 用记事本/VS Code 打开 .env 文件
# Windows:
notepad .env

# Mac/Linux:
nano .env
```

将文件内容修改为：

```env
# 豆包 (Doubao / Ark) API Key
# 获取地址: https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey
ARK_API_KEY=把你的API-Key粘贴到这里
```

**示例**：
```env
ARK_API_KEY=1a8b8f54-7e25-4138-9542-b7465b47791b
```

> ⚠️ **注意**：
> - 不要加引号
> - 不要有空格
> - `ARK_API_KEY=` 后面直接跟 Key

---

## 🐍 第四步：创建 Python 虚拟环境

在项目根目录执行以下命令：

### Windows:

```bash
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
.venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### Mac / Linux:

```bash
# 创建虚拟环境
python3 -m venv .venv

# 激活虚拟环境
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

等待安装完成，看到类似 `Successfully installed ...` 即表示成功。

## ✅ 第五步：验证配置

运行以下命令测试 API Key 和图片生成功能是否配置正确：

```bash
# 确保在项目根目录，且虚拟环境已激活
python scripts/image_gen.py "一只可爱的猫咪，扁平插画风格" -o projects/test/images/ --size 1K --aspect-ratio 16:9
```

**成功标志**：
- 命令行显示图片生成进度
- `projects/test/images/` 目录下出现 `image_001.png` 文件
- 用图片查看器打开能看到一只猫咪插画

如果报错，请检查：
1. `.env` 文件中的 API Key 是否正确
2. 虚拟环境是否已激活（命令行前面有 `(.venv)` 标识）
3. 网络是否正常连接

## 🚀 第六步：开始使用

配置完成！现在你可以：

1. **把这份文档和你的需求一起发给 AI 工具**（如 Kimi、Claude、ChatGPT 等）
2. 告诉 AI：「我想做一个关于 XXX 的 PPT」
3. AI 会引导你完成主题选择、大纲确认、图片生成等步骤
4. 最终会在 `projects/` 目录下生成完整的 HTML PPT 文件
5. 用浏览器打开生成的 `index.html` 即可查看和演示

### 常用快捷键（在 PPT 页面中）

| 按键 | 功能 |
|------|------|
| `←` / `→` 或 `↑` / `↓` | 切换幻灯片 |
| `F` | 全屏演示 |
| `S` | 演讲者模式（看备注和计时器） |
| `T` | 切换主题 |
| `O` | 幻灯片总览网格 |

## 🆘 常见问题

### Q1: 提示 `ARK_API_KEY not found`
**解决**：检查 `.env` 文件是否在项目根目录，且 KEY 拼写正确（`ARK_API_KEY` 不是 `API_KEY`）。

### Q2: 虚拟环境激活失败（Windows）
**解决**：在 PowerShell 中执行 `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`，然后重试。

### Q3: `pip install` 报错
**解决**：尝试升级 pip：`python -m pip install --upgrade pip`，然后重新安装依赖。

### Q4: 图片生成很慢或失败
**解决**：检查网络连接，确认 API Key 未过期，或稍后再试（可能是服务端繁忙）。

### Q5: 如何导出 PPT 为 PNG 图片？
**解决**：确保安装了 Chrome，然后执行：
```bash
./scripts/render.sh projects/你的项目名/index.html all
```
